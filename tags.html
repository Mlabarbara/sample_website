Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML taables via table and related tags.

The tables below do not include all HTML tags, but them so list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div div makes a div? for styling
header header makes a header for styling
